### 第二步
#### 讲解
首先我们学习如何创建一个`set`。

在`main`函数里面通过`set<string> country`来定义一个储存字符串的空的`set`。当然`set`可以存任何类型的数据，比如`set<int> s`等等。在本课中我们用 string 来举例。
```c++
set<string> country;
```

#### 提示
```c++
#include <iostream>
#include <set>
using namespace std;
int main() {
    set<string> country;

    return 0;
}
```

#### 代码
```c++
#include <iostream>
#include <set>
using namespace std;
int main() {
    set<string> country;

    return 0;
}
```
